## Redundant / Deprecated Behaviors

- Deallocation events must not use `PROJECT_ASSIGNMENT`; use `PROJECT_DEALLOCATION`.
- Ad-hoc string interpolation for types is discouraged; use enum constants.
- Scheduler reminder jobs should be idempotent per receiver/day.
